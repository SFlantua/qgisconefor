Conefor Plugin

This plugin integrates Conefor, a software package for studying landscape connectivity into QGIS via the Sextante framework.

Further information available at:

  • Conefor homepage: www.conefor.org
  • Plugin page, with source code and bug tracker: hub.qgis.org
  • Conefor author: Santiago Saura Martinez de Toda
